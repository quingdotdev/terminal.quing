# development

this project is developed on windows with electron and vite. use a current node lts. bun is used in the repository, and npm works as an alternative.

## install

```bash
bun install
```

or with npm.

```bash
npm install
```

## run

```bash
bun run dev
```

## build

```bash
bun run build
```

the packaging script targets windows.

```bash
bun run dist
```

## lint

```bash
bun run lint
```
