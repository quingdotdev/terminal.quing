# development

this project is developed on windows with electron and vite. use a current node lts. bun is the package manager for this repo.

## install

```bash
bun install
```

## run

```bash
bun run dev
```

## build

```bash
bun run build
```

the packaging script targets windows.

```bash
bun run dist
```

## lint

```bash
bun run lint
```
