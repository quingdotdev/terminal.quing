# code of conduct

this project values direct, respectful collaboration. the goal is to build a solid tool without noise.

## scope

this code of conduct applies to issues, pull requests, discussions, and any other project spaces.

## expected behavior

communicate with clarity and respect. focus on the work. avoid harassment, hostility, or personal attacks.

## enforcement

maintainers may edit or remove content that violates this code of conduct. repeat violations can result in being blocked from the project. report problems through github issues.
